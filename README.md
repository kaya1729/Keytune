# Keytune

キーボードで遊べるシンプルな6鍵音ゲーです

# 操作方法

## 起動方法

- フォルダ内にある"Keytune.exe"ファイルを実行してください

## 選曲画面

- gキー、hキー: 曲選択
- dキー、fキー: 難易度選択
- kキー: 設定、選曲の画面切り替え
- spaceキー: ゲームスタート
- escapeキー: タイトル画面に戻る

## ゲーム画面

- d, f, g, h, j, kキーがそれぞれの6レーンと対応しています
- 青いノーツが下のラインに来たらタイミングよくキーをタップ

## 終了方法

- タイトル画面でEscapeキーを押すと終了します
